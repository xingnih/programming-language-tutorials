# LeetCode Language Kits（各語言刷題速成）

這個 repo 的目標很單純：
> 讓你用各種語言在 LeetCode 上「寫得出來、交得上去」。

不講演算法原理，專注在：
- LeetCode 常見函式模板（Solution / function signature）
- 語法必用（if/for/函式/回傳/字串/陣列）
- 常用資料結構速查（map/set/stack/queue/heap）
- 小練習（練寫法，不追求最優解）

## 資料夾
每個語言一個資料夾（內容全是 Markdown）：

- c/
- cpp/
- csharp/
- go/
- java/
- javascript/
- kotlin/
- php/
- python/
- ruby/
- rust/
- scala/
- swift/
- typescript/

## 每個語言的固定結構
- `README.md`：LeetCode 模板 + 讀法
- `basics.md`：語法必用（直接可貼的 snippet）
- `ds-cheatsheet.md`：容器速查（map/set/stack/queue…）
- `practice.md`：小練習
- `solutions.md`：練習參考解（方便對答案/交流）

## 如何貢獻（歡迎 PR）
- 補齊各語言 `basics.md` / `ds-cheatsheet.md` 的「可直接貼上」 snippet
- 把 `practice.md` 題目加上參考解（可另開 `solutions.md`）
- 新增 LeetCode 支援的語言資料夾
