# Programming Language Tutorials (LeetCode Language Kits)

> 目標：讓你用各種語言在 LeetCode 上 **「寫得出來、交得上去」**。

這個 repo 不主打演算法推導，而是把「刷題時最常卡住的語言寫法」整理成速查 + 小練習：

- ✅ LeetCode 常見模板（`Solution` / function signature）
- ✅ 基礎語法必備（if/for/函式/回傳/字串/陣列）
- ✅ 常用資料結構速查（map/set/stack/queue/heap）
- ✅ 小練習（練寫法，不追求最優解）

## Folder structure (per language)

每個語言資料夾固定包含：

- `README.md`：LeetCode 模板 + 使用說明
- `basics.md`：語法速查（偏可直接貼的 snippet）
- `ds-cheatsheet.md`：容器/資料結構速查（map/set/stack/queue…）
- `practice.md`：小練習
- `solutions.md`：練習參考解（對答案/交流用）

---

## How to use

1. 先選一個語言（例如 `python/`）
2. 看 `README.md` 了解 LeetCode 模板
3. 快速過一遍 `basics.md` / `ds-cheatsheet.md`
4. 用 `practice.md` 練手感
5. 卡住再對照 `solutions.md`
